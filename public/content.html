this is a basic example of tiptap. Sure, there are all kind of simple basic text styles you’d probably expect from a text editor. But wait until you see the lists:

  • That’s a bullet list with one …

  • … or two list items.

And here's a numbered list with 2 items:

  1. Item 1

  2. Item 2

Isn’t that great? And all of that is editable. But wait, there’s more. Let’s try a code block:

body {
  display: none;
}

p {
  color: white;
}

I know, I know, this is impressive. It’s only the tip of the iceberg though. And if you need inline code, that's also possible.
Give it a try and click a little bit around. Don’t forget to check the other examples too.
